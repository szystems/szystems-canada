Paginas Web

¿Por qué tener una página web?

La gran visibilidad que ofrece una página web responsive para tu negocio, a la que cualquier usuario accede desde su móvil, Tablet o pc, es una buena razón para que empieces a desarrollar hoy mismo tu sitio web.

Una página web es un escaparate que muestra tus productos e informa sobre tu negocio. Una página web funciona como una plataforma de comunicación.

Gracias a ella, tu marca logra obtener una visibilidad que de otro modo sería imposible. El sitio web de una empresa es el escaparate virtual de sus productos o servicios y su carta de presentación, ya que transmite su identidad.

Además, gracias a la versatilidad que proporcionan las páginas web como soporte, pueden incluir todo tipo de información, siendo algunos ejemplos los siguientes:

  • La historia de la empresa.
  • El catálogo de productos y sus novedades.
  • Las ofertas black friday, etc.

En este sentido, podemos decir que una website actúa como un canal de comunicación altamente efectivo donde no sólo se genera una identidad, sino que además establece un contacto directo con sus usuarios.

¿Qué tipo de página web necesitas?

Dependiendo las necesidades que tu empresa presente, te ofrecemos los siguientes tipos de páginas web para que se acomode fácilmente a tus necesidades, presupuesto o propósitos.

Landing Page

Este tipo de página podría ser la entrada de tu empresa al mundo globalizado y formal con un diseño responsivo para visualizarse en cualquier dispositivo, solo con la información más básica que quieres presentar en una sola pagina dividida en diferentes secciones, además de poder tener comunicación directa a través de las redes sociales integradas en la misma.

Lo que ofrecemos:

- Dominio por 1 año

- Diseño responsivo (vista adecuada para Tablet, celular y pc)

- Hosting de 500MB por 1 Año

- Construidas con lenguaje HTML, PHP, Java Script y Bootstrap

- Una pagina dividida en diferentes secciones

- Formulario de contacto

- Integracion de redes sociales

- 5 correos electrónicos con el nombre del dominio por 1 año

Página web básica

Este tipo de página web a comparación de una landing page ya te ofrece una formalidad mayor en la que tienes repartida tu información en hasta 5 paginas distintas dentro de tu web dándole a los visitantes de la pagina un mejor orden de la información que les quieras dar a conocer.

Lo que ofrecemos:

- Dominio por 1 año

- Diseño responsivo (vista adecuada para Tablet, celular y pc)

- Hosting de 500MB por 1 Año

- Construidas con lenguaje HTML, PHP, Java Script y Bootstrap

- Hasta 5 secciones independientes

- Formulario de contacto

- Integracion de redes sociales

- 5 correos electrónicos con el nombre del dominio por 1 año

Página web Avanzada

Este tipo de pagina web es para el tipo de empresas donde quieren dar a conocer catálogos de productos, menus de restaurantes, servicios, eventos, etc. Ofreciéndole a los visitantes mayor información detallada sobre lo que ofrece la empresa, puedes dividir esta información hasta en 10 paginas distintas dentro de tu web dándole a los visitantes un mejor orden de la información que quieras darles a conocer, además de obtener hosting con espacio ilimitado y correos electrónicos ilimitados para tus trabajadores.

Lo que ofrecemos:

- Dominio por 1 año

- Diseño responsivo (vista adecuada para Tablet, celular y pc)

- Hosting ilimitado por 1 Año

- Construidas con lenguaje HTML, PHP, Java Script, Bootstrap y laravel

- Hasta 10 secciones independientes

- Formulario de contacto

- Integracion de redes sociales

- Correos electrónicos ilimitados con el nombre del dominio por 1 año
